# Brew Heaven Coffee Shop ☕

## 📌 Project Description
Brew Heaven Coffee Shop is a modern, responsive web design project created as part of a Web Technology final requirement. The website represents a fictional coffee shop and includes sections such as a homepage (hero section), menu display, image gallery, and contact form.

The goal of this project is to apply HTML, CSS, and Bootstrap 5 in building a clean, user-friendly, and responsive website that follows best practices in web design.

---

## 🎯 Project Goals
- Create a responsive and visually appealing coffee shop website
- Apply semantic HTML5 structure for better readability and accessibility
- Use Bootstrap 5 for layout, responsiveness, and UI components
- Implement interactive features such as:
  - Navigation bar with mobile toggle
  - Image carousel (gallery section)
  - Contact form with validation
- Ensure the website works well across different screen sizes (mobile-first design)
- Demonstrate proper UI/UX design principles

---

## 🛠️ Technologies Used
- **HTML5** – Structure and content of the website  
- **CSS3** – Custom styling and layout  
- **Bootstrap 5** – Responsive design and pre-built components  
- **JavaScript (Vanilla JS)** – Form validation and interactivity  
- **Unsplash** – Image sources for the website
